1. Visualisation des données

Ici, tu peux visualiser les différents jeux de données génétiques disponibles.

L'objectif par la suite est d'y appliquer un ou plusieurs algorithmes de Machine Learning afin de déterminer les interactions entre gènes et l'évolution de leurs expressions pour une perturbation donnée.

Ce type d'étude est permet de mieux comprendre les phénomènes génétiques


Pour visualiser les données, on utilise des courbes pour les données temporelles et des histogrammes pour les données des états stables (avec ou sans perturbation).

N'hésite pas à filtrer les données affichées pour plus de visibilité

Tu peux aussi retravailler les points de mesures des courbes en cliquant dessus si tu le souhaites !
