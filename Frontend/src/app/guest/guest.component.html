1. Visualisation des données

Ici, tu peux visualiser les différents jeux de données génétiques disponibles.

L'objectif par la suite est d'appliquer un ou plusieurs algorithmes de Machine Learning dessus afin de déterminer les interactions entre les gènes et l'évolution de leurs expressions pour une perturbation donnée.

Ce type d'étude est très intéressante pour mieux comprendre les phénomènes génétiques

Concernant la visualisation des données, il s'agit de courbe pour les données temporelles et d'histogramme pour les données des états stables (avec ou sans perturbation).

N'hésite pas a filtrer les données affichées pour plus de visibilité

Tu peux aussi retravailler des points de mesures des courbes en cliquant dessus si tu le souhaites !
