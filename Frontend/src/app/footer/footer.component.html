A propos de nous

Ce site a été développé dans le cadre du programme Centrale Digital Lab de l'Ecole Centrale de Nantes.

Les trois étudiants en Machine Learning Gauthier Lacroix, Erick Ah-Mouck et Alexandre Féron ont élaboré, en colaboration avec les chercheurs de l'équipe MeForBio du LS2N, cette interface pour permettre d'exploiter facilement leurs algorithmes d'apprentissage.

Ce site est basé sur le challenge d'exploitation de données génétiques DREAM4.

Pour plus d'informations, tu disposes de pointeurs sur les différentes parties prenantes de ce projet.

Le code source de ce travail est disponible sous licence MIT sur dépot Github.
