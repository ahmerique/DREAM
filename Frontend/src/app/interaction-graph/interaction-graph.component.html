4. Prédiction du graph d'interaction

Suite à l'apprentissage, le modèle a determiné les influences entre les gènes.

Ces influences sont représentées par un graphe d'interaction.

Chaque liaison entre 2 gènes correspond à un lien. La différence de couleur correspond à l'action du gène : s'il a une action d'inhibition, il sera en rouge et s'il a une action d'activation, il sera en vert. Si tu double-cliques sur un gène, tu pourras mettre en valeur tous les liens de ce gène avec d'autres. Dans ce cas là, la différence de couleur marquera le sens de l'interaction et non l'action du gène.

4. Prediction on the interaction graph

After the learning, the model determined the interactions between the genes. Those interaction are represented using an interaction graph. Each bond between two genes represent a link between those genes. The color difference represent the action of the gene: If there's an inhibition, the link will be red. If there's an activation, the link will be green. If you double-click on a gene, you will highlight all the bonds of the specified gene. In that case, the color difference will represent the direction of the interaction instead of its action.
